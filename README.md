# todo-ts

*簡略說明

1. 使用Vue-Cli建立專案，用TypeScript和Vuex來撰寫todo list app。
2. CSS使用Tailwind(安裝指令vue add tailwind)

*設定
1. shims-vue.d.ts檔案是Ambient Declarations(外部模組定義)，給專案內的vue檔案做模組宣告，讓編輯器能辨識且使vue框架等函式庫的功能能融入TypeScript。
